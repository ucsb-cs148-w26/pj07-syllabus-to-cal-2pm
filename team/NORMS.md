# Norms
### please follow these

1. If you can't make it to lecture or section please let the team know.
2. Don't go ghost.
3. Start assginments early, don't do last minute messaging. 
4. Make sure to ask for help when you need it we are all here to help.
5. When Pull Requests go live make sure to review quickly to merge so they dont stack. 
6. If you have any issues, bring them up. 
7. Branch format should be **FirstAndLastInitials-branch_name**
